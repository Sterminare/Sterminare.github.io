Analog Voltmeter Clock


This is a clock made from three analog voltmeters. I used 3 5v voltmeters, of which I scanned the original cover plates, edited them to display information about the hour, minute, and seconds, printed and reassembled with the new covers. Everything is controlled by an Arduino Nano with a Real Time Clock module, so it keeps time even when it is not connected to power.

If I were to make another version of this clock, I would make a couple changes. I would omit the seconds display, because while cool to watch, it takes up a lot of room. For the same reason, I would put the hours and minutes stacked vertically so it takes up less space on a desk. I would also make it battery powered so it wouldn't require wall power.
