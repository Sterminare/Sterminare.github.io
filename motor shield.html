Motor Shield

Purpose

This motor shield was designed to streamline the process of creating a simple robot, by removing complex wiring to enable more focus on the design and programming of simple robots. It is designed to mount on a generic Arduino Uno board.

Features

The shield features an L239D H-bridge motor driver and four servo headers, with flexible, polarity-protected power supply options for each. It also has a header for an HM-10 bluetooth module for wireless control. It has space for a mini breadboard to be mounted directly onto it, which provides a convenient place for additional circuits + sensors to be wired if needed.
